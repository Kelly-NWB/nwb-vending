LLM API Bootcamp

ENTRY: mf-train-001 · SECTION: training · PROVIDER: xAI Grok API

Theory-first interactive course (~2 hours). Maps console → request → credits → product picker. Built for developers and agents wiring automation without drowning in docs.

Problem

You have API credits and a key but no mental model: what is an API vs browser chat, how requests work, what drains credits, which product (Chat, Build, Voice, Imagine) fits which job.

What's in the full build

  1. What an API actually is
  2. Your xAI console decoded
  3. Anatomy of a request (HTTP, JSON, PowerShell walkthrough)
  4. Credit math and cost estimator
  5. Capabilities beyond plain chat (tools, voice, batch)
  6. Troubleshooting and printable one-pager

Includes checklists, decision tree, live cost calculator, progress tracking (localStorage).

Free preview — Module 1 opener

API in one sentence: A remote service your software calls with a fixed question/answer format — so a program gets work done without a human clicking buttons.

Browser chat = you typing. API = your program calling the same models 24/7. Same models, different door.

When you don't need the API: one-off questions, brainstorming — use the browser.

When you do: Discord bot, batch narration, app button, cron job, voice pipeline.
