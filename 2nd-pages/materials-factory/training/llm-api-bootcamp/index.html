Free preview · Module 1 opener

API in one sentence: A remote service your software calls with a fixed question/answer format so a program gets work done without a human clicking buttons.

Browser chat = you typing. API = your program calling the same models 24/7. Same models, different door.

When you don't need the API: one-off questions, brainstorming. Use the browser.

When you do: Discord bot, batch narration, app button, cron job, voice pipeline.
