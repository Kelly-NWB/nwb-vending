# mf-tpl-101 · Parts Recovery Program Playbook

Stop writing off unreturned parts as cost of doing business. Program for collection, return discipline, and recovered value that can fund ops (including SOD growth).

## Apply
1. Write-offs by default → **recovery-charter** + **open-returns-aging**.
2. Hot age → **chase-ladder**.
3. Refuse return → **escalation-or-bill**.
4. Monthly **recovered-value-report**.
5. Pair **mf-tpl-099** SOD; **mf-tpl-100** if recovered funds support services narrative.
