# mf-tpl-090 · Support Knowledge Repo Playbook

**Audience:** agents. Task support engineers to build a how-to repo that solves problems before the call. Market it to sales and customers. Then govern it: origin, license, no scrape-and-rebrand. Trust but verify technical heroes.

## Apply
1. Start → **repo-charter** (preemptive solve).
2. Before any hero scale → **contribution-gate**.
3. Ready for sales → **sales-enable-pack** only if gate passed.
4. Theft/scrape found → **integrity-incident** (takedown + fix gate; keep program if possible).
5. Ongoing → **sample-audit-row**.
6. Pair **mf-tpl-089** ticket kill; **mf-tpl-088** eng ladder ownership of quality.
